ResponseInterface request(array $cmd = [], string $path = "")

Perform API request using the given command.

The IBS/Moniker platform exposes many endpoints under one host, where the path selects the operation (e.g. Domain/Create, Domain/Info). The base host is configured on the SocketConfig (liveUrl/oteUrl, host only, with a trailing slash); the per-operation path is appended by {\CNIC\AbstractClient::performRequest()} and therefore must be supplied per request.

Parameters

array $cmd

API command
string $path

path segment appended to the base URL to select the endpoint
